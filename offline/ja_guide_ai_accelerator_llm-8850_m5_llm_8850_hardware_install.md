# LLM-8850 Card ハードウェアインストール

ソース: https://docs.m5stack.com/ja/guide/ai_accelerator/llm-8850/m5_llm_8850_hardware_install

Raspberry Pi 5 に LLM-8850 アクセラレータカードを取り付けるには、以下のいずれかの拡張ボードを使用します。

> **警告:** LLM-8850 Card を取り付ける前に、**必ず Raspberry Pi 5 の電源を切断してください**。通電状態での取り付けは故障の原因となり、厳禁です。

## 方法1：LLM-8850 PiHat 拡張ボードを使用する場合

### 電源に関する注意事項

LLM-8850 PiHat 拡張ボードを使用する場合は、以下の点に注意してください：

- 拡張ボードの **PD 給電インターフェース** を通じてシステム全体に給電してください
- 必要な給電能力：**9V@3A（27W）以上**

### 取り付け手順

取り付けの詳細な手順については、公式サイトのインストール動画をご参照ください。

## 方法2：M.2 HAT+ 拡張ボードを使用する場合

### 電源に関する注意事項

M.2 HAT+ 拡張ボードを使用する場合は、**DC 5V@3A** の給電能力を持つスイッチング電源アダプタの使用を推奨します。

> **注意:** PD（Power Delivery）プロトコル対応の電源アダプタは使用しないでください。PD電源を使用した場合、プロトコルのネゴシエーションにより最大出力が得られず、デバイスの動作に異常をきたす可能性があります。
