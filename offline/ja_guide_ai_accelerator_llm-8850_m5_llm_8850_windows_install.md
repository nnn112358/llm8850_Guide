# LLM-8850 Card Windows 環境設定

ソース: https://docs.m5stack.com/ja/guide/ai_accelerator/llm-8850/m5_llm_8850_windows_install

## Windows バージョン要件

| Windows | バージョン |
|---------|-----------|
| 10      | 22H2      |
| 11      | >= 23H2   |

`Win+R` を押して `winver` と入力すると、現在の Windows バージョン情報を確認できます。

> **注意:** Windows 10 64ビット および Windows 11 64ビット オペレーティングシステムのみサポートしています。

`Win+I` を押して設定 > 更新とセキュリティ > Windows Update を開き、Windows バージョンを自動更新してください。Windows 10 の場合は、[Microsoft 更新アシスタント](https://www.microsoft.com/software-download/windows10) を使用して 22H2 バージョンへの更新を推奨します。

また、スリープ機能を無効にしてください。`Win+I` を押して設定 > システム > 電源とスリープ を開き、スリープを **なし** に変更します。

## 環境準備

- セキュリティソフトを無効にすることを推奨します（誤検知によるブロックを回避するため）。
- Visual Studio 2022 ランタイムライブラリ `VC_redist.x64.exe` をダウンロードし（[Microsoft ダウンロードリンク](https://learn.microsoft.com/cpp/windows/latest-supported-vc-redist)）、管理者権限で `VC_redist.x64.exe` をインストールしてください。
- シャットダウンして電源を切った後に、デバイスをマザーボードに接続してください。**通電状態での接続・取り外しは厳禁です。**
- 起動後、`Win+R` を押して `devmgmt.msc` と入力し、デバイスマネージャーを開きます。「その他のデバイス」に不明なマルチメディアビデオコントローラーが表示されます。プロパティを開くと、デバイスの VENDOR が `1F4B` であることを確認できます。

## ドライバインストール

> **注意:** インストールおよびアンインストールには管理者権限が必要です。また、システムのスリープ機能を事前に無効にしてください。システムに旧バージョンの AXCL がインストールされている場合は、先にアンインストールしてから新しいバージョンをインストールしてください。

Windows リリースパッケージ `axcl_win64_setup_Vx.x.x_yyyymmdd_NOxxxx.exe` は、ドライバ、ダイナミックリンクライブラリ（インポートライブラリ含む）、実行可能プログラム（axcl-smi.exe 等）、およびサンプルソースコードを自動的にインストールします。管理者権限でリリースパッケージを実行し、ウィザードの指示に従ってインストールを完了してください。

1. ダウンロードをクリックします。
2. インストールをクリックします。
3. すべて選択してインストールすることを推奨します。
4. インストールパスを選択してください。このディレクトリには実行可能プログラム、アンインストールプログラム、ヘッダファイル、およびダイナミックライブラリが格納されます。
5. インストールが完了するのを待ち、コンピュータを再起動します。

再起動後、`Win+R` を押して `devmgmt.msc` と入力してデバイスマネージャーを開くと、「システムデバイス」に Axera NPU Accelerator Device が表示されます。

次に、インストールパス `AXCL\axcl\out\axcl_win_x64\bin` ディレクトリに移動し、Windows Terminal で `axcl-smi.exe` を実行して動作を確認してください。

## ドライバのアンインストール

インストールパス AXCL に移動し、`uninst.exe` を実行するとアンインストールできます。

## プログラムのコンパイル

> **注意:** AXCL Windows SDK は Visual Studio 2022 / Visual Studio 2026 ツールチェーンでコンパイルできます。必要な環境依存関係は以下の通りです：

- [Visual Studio Community 2026](https://visualstudio.microsoft.com/) - [公式ドキュメント](https://learn.microsoft.com/visualstudio/install/install-visual-studio)を参考にしてください。必須コンポーネント：「C++ によるデスクトップ開発」
- CMake - バージョン 3.20 以上
- ninja - v1.31.1 を推奨。`ninja.exe` のパスをシステム環境変数 Path に追加してください。
- Python3 - conda 経由でインストール可能です。

環境のインストールが完了したら、インストールパス `AXCL\axcl\scripts` に移動し、`build_win64.cmd` を実行してコンパイルします。

コンパイル出力ディレクトリに移動し、以下のコマンドを実行します（実際のモデルファイルパスに置き換えてください）：

```bash
axcl_run_model.exe -m yolo11s.axmodel -r 10
```
